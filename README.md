# Utility functions for Linear Algebra

## Usage
```
v = Vector([1, 2, 3])

v.add(Vector([4, 5, 6]))

v.minus(Vector([4, 5, 6))

v.scalar_multiply(2)

v.magnitude()

v.normalize()

v.dot(Vector([4, 5, 6]))

v.angle(Vector([4, 5, 6]))

v.is_parallel_to(Vector([4, 5, 6]))

v.is_orthogonal_to(Vector([4, 5, 6]))
```
